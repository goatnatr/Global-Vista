# Global Vista Master Archive

## Purpose

This folder contains historical Global Vista materials that are retained for reference but are not part of the active development baseline.

Examples include:

- Frozen release baselines
- Historical documentation
- Superseded standards
- Legacy project structures
- Previous repository snapshots
- Archived research

## Policy

Files in this folder are retained for historical reference.

Active development SHALL occur outside this folder unless a historical version is being restored or examined.
